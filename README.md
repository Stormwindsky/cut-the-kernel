# Cut the Kernel

**Cut the Kernel** is an unofficial, lightweight desktop client for playing the YouTube Playables version of *Cut the Rope* on Linux. Built with Electron, it provides a native-like experience for this classic puzzle game.

## Features

- 🐧 **Linux First**: Designed specifically for Linux environments.
- 📦 **Isolated**: Runs in a sandboxed environment for better security.
- 🚀 **Fast**: Minimalist overhead, focusing only on the game.
- 🔍 **Searchable**: Optimized with keywords like "CTR" and "Cut the rope" for Linux app launchers.

## How to Run (Development)

Ensure you have [Node.js](https://nodejs.org/) installed on your system.

1. Clone the repository:
   ```bash
   git clone https://github.com/Stormwindsky/cut-the-kernel.git
   cd cut-the-kernel
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```

## Building as AppImage

To compile the application into a standalone `.AppImage` file:

```bash
npm run dist
```
